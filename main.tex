\section{Programme Overview}

\subsection{Target Effort}

The objective of each working set is to finish with the prescribed target effort for that section of the programme. Unless otherwise specified, this is approximately one repetition in reserve (1 RIR).

Warm-up sets are intended only to prepare for the working sets and should not produce significant fatigue.

\subsection{Training Philosophy}

This programme is designed to maximise long-term strength and muscle development while remaining simple, repeatable and sustainable under normal commercial gym conditions. Load is increased only when the prescribed performance criteria are achieved rather than according to a fixed timetable.

The primary strength programme forms the foundation of long-term progression. An alternative hypertrophy block is provided for occasions when training time is limited or when a simpler hypertrophy-focused session is preferred. All alternative work is derived from the main-programme starting working weights to ensure both variants remain aligned.

Consistency of technique, objective progression, and long-term adherence take priority over rapid increases in load. Every increase in weight should be earned through demonstrated performance rather than attempted prematurely.

\subsection{Key Definitions}

The following terms are used throughout this document.

\begin{description}
\item[Starting working weight:] The load used for the first working set of a main lift. Where work-set increments are prescribed, subsequent working sets are performed by increasing the load from this starting weight.

\item[Warm-up:] Preparatory sets performed before the working sets to prepare the muscles, joints and movement pattern without causing significant fatigue.

\item[Working set:] A set intended to provide the prescribed training stimulus and to which the target effort applies.

\item[Repetitions in Reserve (RIR):] The estimated number of additional repetitions that could have been completed while maintaining acceptable technique. Unless otherwise specified, all working sets should finish with approximately one repetition in reserve.

\item[Main starting working weight:] The current starting working weight prescribed for the first working set of the corresponding exercise in the Main Strength Block. This value is used when calculating loads for the Alternative Hypertrophy Block.

\item[Work-set increment:] The increase in load applied between successive working sets where specified.

\item[Session progression:] The increase in starting working weight applied between training sessions after the prescribed progression criteria have been met.
\end{description}

\section{Using this Manual}

Each lookup table lists the current prescribed working weight together with several future progressions.

For every exercise:

\begin{itemize}
\item Circle the row corresponding to the current starting working weight.
\item Continue using that row until the progression criteria have been met.
\item When the exercise is progressed, move to the next row and circle it.
\item Leave future rows unmarked until they become active.
\end{itemize}

For the Alternative Hypertrophy Block, first select the current starting working weight row and then the current percentage column. The value at their intersection is the prescribed working weight.

\section{Warm-up \& Accessory Block}

Target effort: Finish every working set with approximately three repetitions in reserve (3 RIR). The purpose of this block is to increase body temperature, rehearse movement patterns and provide a small maintenance stimulus without impairing performance during the Main Strength Block.

Each exercise lists its prescribed work and working-weight progression. Circle the current row and advance to the next row only when the progression criteria have been satisfied.

\section{Main Strength Block}

Target effort: Finish every working set with approximately one repetition in reserve (1 RIR).

Bench press and back squat should not be taken to technical failure, especially where safety bars or a competent spotter are not available. In those circumstances, stop earlier than the target effort if required to preserve safety.

For Deadlift, Bench Press and Back Squat, the starting working weight is the load used for the first of the three working sets. Increase the load by the prescribed work-set increment after each working set.

Circle the current row and advance to the next row only when the progression criteria have been satisfied.

Barbell loading tables list the total working weight and the plates to be placed on each side of the bar. For example, ``20\,kg'' in the plates-per-side column means one 20\,kg plate on each side of the bar.

For Deadlift, the minimum floor-pull load is the lightest listed deadlift working weight for the relevant bar: 25\,kg with the 20\,kg bar and 20\,kg with the 15\,kg bar.

When training in pairs, rest while the other lifter completes their set. If technique or performance deteriorates because rest periods are too short, take additional rest before the next working set.

No additional warm-up sets are normally required for the remaining exercises because they are preceded by the Warm-up \& Accessory Block and progressively heavier compound lifts. Perform an additional light familiarisation set only if an exercise does not feel adequately prepared.

\section{Progression Rules}

\subsection{Primary 3\texorpdfstring{$\times$}{x}5 Lifts}

\begin{itemize}
\item Increase the load between working sets by the prescribed work-set increment.
\item For Deadlift, Bench Press and Back Squat, this gives three working sets: the starting working weight, the starting working weight plus one increment, and the starting working weight plus two increments.
\item The final working set is expected to be closest to the target effort. The first and second working sets may be easier.
\item Progression is earned only when every prescribed repetition is completed using the intended range of motion and acceptable technique. Altered movement patterns, shortened range of motion or loss of control do not count towards progression.
\item If the final working set finishes with two or more repetitions in reserve (2 RIR) and all working sets were completed with acceptable technique, increase the starting working weight for the next session by one progression step.
\item If the final working set finishes with approximately one repetition in reserve (1 RIR), retain the same starting working weight.
\item If a working set reaches technical failure, or acceptable technique is lost before all prescribed repetitions have been completed, apply the regression rule.
\end{itemize}

\subsection{Regression Rules}

Regression is used when the prescribed load is too heavy to complete with the intended technique, range of motion and target effort.

\begin{itemize}
\item A set does not count as successfully completed if the intended range of motion is shortened, control is lost, or the movement pattern changes substantially in order to complete the repetition.
\item If a main lift reaches technical failure, or acceptable technique is lost before all prescribed repetitions have been completed, reduce the starting working weight by one progression step for the next session.
\item If a primary lift is completed with acceptable technique but finishes with zero repetitions in reserve (0 RIR), retain the same starting working weight for the next session. If this happens in two consecutive sessions, reduce the starting working weight by one progression step.
\item If the same main lift requires regression in two consecutive sessions, reduce the starting working weight by one additional progression step after the second session, giving a total reduction of two progression steps from the original failed load.
\item For accessory exercises, reduce the working weight by one progression step if the prescribed repetitions or distance cannot be completed with acceptable technique.
\item For cable exercises loaded with kettlebells, reduce to the previous kettlebell if the prescribed repetitions cannot be completed with the intended movement pattern and control.
\item For carries, reduce to the previous kettlebell if posture, shoulder position, walking control or grip position cannot be maintained over the prescribed distance.
\item Do not compensate for excessive load by using shortened range of motion, momentum, bouncing, twisting, excessive back extension, or rushing the set.
\item If an exercise is already at the lowest listed working weight, retain that weight and reduce effort, range of loading, or total work as required to preserve acceptable technique.
\end{itemize}

\subsection{Secondary and Accessory Progression}

For all secondary and accessory exercises, retain the current working weight unless the progression or regression criteria are met. Progress only when the exercise is clearly easier than the target effort. Regress when the prescribed work cannot be completed with acceptable technique.

The target effort remains approximately 3 RIR unless stated otherwise; 4 RIR is the threshold for increasing load, not the normal target.

\subsubsection{Standing Calf Raise}

\begin{itemize}
\item Move to the next kettlebell when every set finishes with 4 RIR or more.
\end{itemize}

\subsubsection{Suitcase Carry}

\begin{itemize}
\item Move to the next kettlebell when both sets on each side are completed over the prescribed distance while maintaining an upright torso, level shoulders and a normal walking pace, and the set does not feel close to technical breakdown.
\end{itemize}

\subsubsection{Alternating Supinating Dumbbell Curl}

\begin{itemize}
\item Move to the next dumbbell when every set finishes with 4 RIR or more.
\end{itemize}

\subsubsection{Cable Crunch}

\begin{itemize}
\item Move to the next kettlebell when every set finishes with 4 RIR or more.
\end{itemize}

\subsection{Alternative Block Progression}

For Deadlift, Bench Press and Back Squat:

\begin{itemize}
\item Start at 70\% of the main-programme starting working weight.
\item Use a lower percentage column if 70\% is too heavy.
\item The available percentage range is 50--90\%, in five-percentage-point increments.
\item If every set finishes with 4 RIR or more, increase the percentage by five percentage points for the next Alternative Hypertrophy Block session.
\item If every set finishes with 2--3 RIR, retain the same percentage for the next Alternative Hypertrophy Block session.
\item If any set reaches 0 RIR but all repetitions are completed with acceptable technique, decrease the percentage by five percentage points for the next Alternative Hypertrophy Block session.
\item If any set reaches technical failure or technique breaks down before all prescribed repetitions are completed, decrease the percentage by ten percentage points for the next Alternative Hypertrophy Block session.
\item Do not progress beyond 90\%. If 90\% becomes too easy, continue at 90\% or return to the Main Strength Block.
\item Calculate the target load using the current percentage of the main-programme starting working weight, then round to the nearest load that can be assembled using the available plates. If the calculated load is exactly halfway between two achievable loads, round down.
\end{itemize}

The values in the percentage tables are total bar loads, not plates per side.

Lat Pulldown, Pendlay Row and Strict Press use the same working-weight rows as the Main Strength Block.

\section{Plate Calculations}

Unless otherwise stated, all prescribed barbell loads refer to the total load on the bar.

In barbell loading tables, the plates-per-side column specifies the plates to be placed on each side of the bar.

For example,

\begin{itemize}
\item 60\,kg total load with a 20\,kg bar requires 20\,kg per side.
\item 35\,kg total load with a 15\,kg bar requires 10\,kg per side.
\end{itemize}

Where percentage-based loads are prescribed, calculate the target load using the main-programme starting working weight, then round to the nearest load that can be assembled using the available equipment. If the calculated load is exactly halfway between two achievable loads, round down.
