\section{Programme Overview}

\subsection{Target Effort}

The objective of every exercise in this programme is to complete each working set with approximately one repetition in reserve (1 RIR). This provides a high level of training stimulus while allowing technique to remain consistent and recovery to be managed effectively.

Warm-up sets are intended only to prepare for the working sets and should not produce significant fatigue.

\subsection{Training Philosophy}

This programme is based on objective, performance-driven progression. Load is increased only when the prescribed performance criteria are achieved rather than according to a fixed timetable.

The primary strength programme forms the foundation of long-term progression. An alternative hypertrophy block is provided for occasions when training time is limited or when additional hypertrophy-focused work is preferred. All alternative work is derived from the main-programme working weights to ensure both variants remain aligned.

Consistency of technique, controlled progression, and long-term adherence take priority over rapid increases in load.

\subsection{Key Definitions}

The following terms are used throughout this document.

\begin{description}
\item[Working weight:] The load used for the first working set of a main lift. Where work-set increments are prescribed, subsequent working sets are performed by increasing the load from this starting weight.

\item[Warm-up:] Preparatory sets performed before the working sets to prepare the muscles, joints and movement pattern without causing significant fatigue.

\item[Working set:] A set intended to provide the prescribed training stimulus and to which the target effort applies.

\item[Repetitions in Reserve (RIR):] The estimated number of additional repetitions that could have been completed while maintaining acceptable technique. Unless otherwise specified, all working sets should finish with approximately one repetition in reserve.

\item[Main-programme working weight:] The current working weight prescribed for the first working set of the corresponding exercise in the Main Strength Block. This value is used when calculating loads for the Alternative Hypertrophy Block.

\item[Work-set increment:] The increase in load applied between successive working sets where specified.

\item[Session progression:] The increase in starting working weight applied between training sessions after the prescribed progression criteria have been met.
\end{description}

\section{Using this Manual}

Each lookup table lists the current prescribed working weight together with several future progressions.

For every exercise:

\begin{itemize}
\item Circle the row corresponding to the current working weight.
\item Continue using that row until the progression criteria have been met.
\item When the exercise is progressed, move to the next row and circle it.
\item Leave future rows unmarked until they become active.
\end{itemize}

For the Alternative Hypertrophy Block, first select the current working-weight row and then the current percentage column. The value at their intersection is the prescribed working weight.

\section{Progression Rules}

\subsection{Main Lifts}

\begin{itemize}
\item Increase the load after each working set by the prescribed work-set increment.
\item If every working set finishes with two or more repetitions in reserve (2 RIR), increase the working weight for the next session by the same increment.
\item If any working set finishes with one repetition in reserve (1 RIR) or fewer, retain the same working weight for the next session.
\end{itemize}

\subsection{Main Lift Exceptions}

\subsubsection{Kathi's Deadlift}

\begin{itemize}
\item Increase by +2$\times$2.5\,kg plates until the working weight reaches a 15\,kg bar with 2$\times$25\,kg plates.
\item Thereafter, increase by +2$\times$5\,kg plates.
\end{itemize}

\subsection{Accessory Progression}

\subsubsection{Standing Calf Raise}

\begin{itemize}
\item Move to the next kettlebell when every set finishes with 2 RIR or more.
\end{itemize}

\subsubsection{Suitcase Carry}

\begin{itemize}
\item Move to the next kettlebell when both sets on each side finish with 2 RIR or more.
\end{itemize}

\subsubsection{Alternating Supinating Dumbbell Curl}

\begin{itemize}
\item Move to the next dumbbell when every set finishes with 2 RIR or more.
\end{itemize}

\subsubsection{Cable Crunch}

\begin{itemize}
\item Move to the next kettlebell when every set finishes with 2 RIR or more.
\end{itemize}

\subsubsection{Lat Pulldown}

\begin{itemize}
\item Move to the next kettlebell when every set finishes with 2 RIR or more.
\end{itemize}

\subsubsection{Pendlay Row}

\begin{itemize}
\item Increase by +2$\times$1.25\,kg plates when every set finishes with 2 RIR or more.
\end{itemize}

\subsubsection{Strict Press}

\begin{itemize}
\item Increase by +2$\times$1.25\,kg plates when every set finishes with 2 RIR or more.
\end{itemize}

\subsection{Alternative Block Progression}

\begin{itemize}
\item Start at 65\% of the main-programme working weight.
\item If every set finishes with 4 RIR or more, increase the percentage by five percentage points for the next Alternative Hypertrophy Block session.
\item If every set finishes with 2--3 RIR, retain the same percentage for the next Alternative Hypertrophy Block session.
\item If any set finishes with 1 RIR or fewer, decrease the percentage by five percentage points for the next Alternative Hypertrophy Block session.
\item Calculate the target load using the current percentage of the main-programme working weight, then round to the nearest load that can be assembled using the available plates, dumbbells, kettlebells or machine increments.
\end{itemize}

\subsection{Heavy-lifting Transition}

Once the specified working weight has been reached, replace the single warm-up set with:

\item Bench Press: 20\,kg bar + 2$\times$20\,kg plates

\end{itemize}

before completing the prescribed working sets.

\section{Plate Calculations}

Unless otherwise stated, all prescribed loads refer to the total load on the bar.

For example,

\begin{itemize}
\item 20\,kg bar + 2$\times$20\,kg plates = 60\,kg total load.
\item 15\,kg bar + 2$\times$10\,kg plates = 35\,kg total load.
\end{itemize}

Where percentage-based loads are prescribed, calculate the target load using the main-programme working weight, then round to the nearest load that can be assembled using the available equipment.
