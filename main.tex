\section{Programme Overview}

\subsection{Target Effort}

The objective of each working set is to finish with the prescribed target effort for that section of the programme. Unless otherwise specified, this is approximately one repetition in reserve (1 RIR).

Warm-up sets are intended only to prepare for the working sets and should not produce significant fatigue.

\subsection{Training Philosophy}

This programme is designed to maximise long-term strength and muscle development while remaining simple, repeatable and sustainable under normal commercial gym conditions. Load is increased only when the prescribed performance criteria are achieved rather than according to a fixed timetable.

The primary strength programme forms the foundation of long-term progression. An alternative hypertrophy block is provided for occasions when training time is limited or when a simpler hypertrophy-focused session is preferred. The Alternative Hypertrophy Block uses the Main Strength Block starting working weights for percentage-based lifts. Non-percentage lifts use their own loading tables so that both variants remain aligned without forcing inappropriate percentage calculations.

Consistency of technique, objective progression, and long-term adherence take priority over rapid increases in load. Every increase in weight should be earned through demonstrated performance rather than attempted prematurely.

\subsection{Key Definitions}

The following terms are used throughout this document.

\begin{description}
\item[Starting working weight:] The load used for the first working set of an exercise. Where work-set increments are prescribed, subsequent working sets are performed by increasing the load from this starting weight.

\item[Warm-up:] Preparatory sets performed before the working sets to prepare the muscles, joints and movement pattern without causing significant fatigue.

\item[Working set:] A set intended to provide the prescribed training stimulus and to which the target effort applies.

\item[Repetitions in Reserve (RIR):] The estimated number of additional repetitions that could have been completed while maintaining acceptable technique.

\item[Main starting working weight:] The current starting working weight prescribed for the first working set of the corresponding exercise in the Main Strength Block. This value is used when calculating percentage-based loads for the Alternative Hypertrophy Block.

\item[Work-set increment:] The increase in load applied between successive working sets where specified.

\item[Session progression:] The increase in working weight or percentage applied between training sessions after the prescribed progression criteria have been met.
\end{description}

\section{Using this Manual}

Each lookup table lists the current prescribed working weight together with several future progressions.

For every exercise:

\begin{itemize}
\item Circle the row corresponding to the current starting working weight.
\item Continue using that row until the progression criteria have been met.
\item When the exercise is progressed, move to the next row and circle it.
\item Leave all future rows unmarked until they become active.
\end{itemize}

For the Alternative Hypertrophy Block, first select the current starting working weight row and then the current percentage column. The value at their intersection is the prescribed working weight.

A set counts as successfully completed only when every repetition is performed with the intended range of motion, control and movement pattern. Shortened range of motion, loss of control, excessive momentum, bouncing, twisting or excessive back extension do not count towards progression.

%=========================================================
\section{Main Strength Block}

Bench Press and Back Squat should not be taken to technical failure, especially where safety bars or a competent spotter are not available. In those circumstances, stop earlier than the target effort if required to preserve safety.

Circle the current row and advance to the next row only when the progression criteria for that exercise have been satisfied.

Barbell loading tables list the total working weight and the plates to be placed on each side of the bar. For example, ``20\,kg'' in the plates-per-side column means one 20\,kg plate on each side of the bar.

For Deadlift, the minimum floor-pull load is the lightest listed Deadlift working weight for the relevant bar: 25\,kg with the 20\,kg bar and 20\,kg with the 15\,kg bar.

When training in pairs, rest while the other lifter completes their set. If technique or performance deteriorates because rest periods are too short, take additional rest before the next working set.

The opening exercises provide general preparation and a small maintenance stimulus while leaving sufficient capacity for the subsequent compound lifts. Deadlift, Bench Press and Back Squat use the specific warm-up sequences listed in their own sections. For the remaining exercises, no additional warm-up sets are normally required. Perform an additional light familiarisation set only if an exercise does not feel adequately prepared.

\subsubsection{Standing Calf Raise}

Work: 2$\times$10

Target effort: Finish each set with approximately three repetitions in reserve (3 RIR).

Progression: Move to the next dumbbell pair when every set finishes with 4 RIR or more.

Retention: Keep the same dumbbell pair when every set finishes with approximately 1--3 RIR.

Regression: Move to the previous dumbbell pair if the prescribed repetitions cannot be completed with acceptable technique.

Technique: Stand upright while holding one dumbbell in each hand. Keep the knees slightly unlocked but do not deliberately bend and straighten them during the repetition. Raise the heels as high as possible, pause briefly at the top, then lower under control until the calves are stretched. Use the largest comfortable range of motion on every repetition. Do not bounce out of the bottom position, cut the top position short, or turn the movement into a squat.

Recommended starting working weight for an average recreational gym-goer with previous resistance-training experience: 22.5\,kg dumbbells in each hand.

\subsubsection{Suitcase Carry}

Work: 2$\times$25\,m per side

Target effort: Complete each carry with stable posture and without approaching technical breakdown.

Progression: Move to the next kettlebell when both sets on each side are completed over the prescribed distance while maintaining an upright torso, level shoulders and a normal walking pace. After reaching the 44\,kg kettlebell, progress using the distance-progression table instead of increasing load.

Retention: Keep the same kettlebell and distance when all carries are completed but posture, walking control or grip position are close to the limit of acceptable technique.

Regression: Move to the previous kettlebell if posture, shoulder position, walking control or grip position cannot be maintained over 25\,m. When using the distance-progression table, move to the previous distance row if posture, shoulder position, walking control or grip position cannot be maintained over the prescribed distance.

Technique: Hold the kettlebell at one side with a neutral grip and stand upright before beginning to walk. Brace the trunk, keep the shoulders level, and allow the arms to hang naturally without shrugging. Walk at a normal, comfortable pace using short, controlled steps while keeping the torso upright and avoiding excessive side-to-side movement. Turn carefully at the end of the prescribed distance before continuing if required. Do not lean towards or away from the kettlebell, use the free arm to brace against the body, run, or continue once grip or posture begins to fail.

Recommended starting working weight for an average recreational gym-goer with previous resistance-training experience: 24\,kg kettlebell.

\paragraph{Distance progression after 44\,kg}

Once 44\,kg can be carried for 2$\times$25\,m per side with stable posture and normal walking control, keep the same kettlebell and increase the prescribed distance by 25\,m per side. Do not reduce the load automatically when distance increases; the purpose of this progression is to extend the same heavy unilateral carry stimulus. If a longer distance cannot be completed with acceptable posture, return to the previous distance row.

The upper bound is 100\,m per side. Beyond this point, the exercise becomes increasingly time-consuming and conditioning-dominant rather than serving its main role in this programme: grip strength, trunk stiffness, frontal-plane control and loaded gait.

\subsubsection{Alternating Supinating Dumbbell Curl}

Work: 2$\times$10

Target effort: Finish each set with approximately three repetitions in reserve (3 RIR).

Progression: Move to the next dumbbell when every set finishes with 4 RIR or more.

Retention: Keep the same dumbbells when every set finishes with approximately 1--3 RIR.

Regression: Move to the previous dumbbell if the prescribed repetitions cannot be completed with acceptable technique.

Technique: Stand upright with one dumbbell in each hand and the arms fully extended. Curl one dumbbell at a time, beginning with the palm facing the body and gradually rotating the forearm so that the palm faces upwards near the top of the movement. Keep the upper arm close to the torso throughout the repetition and lower the dumbbell under control until the elbow is fully extended before beginning the next repetition on that side. Alternate arms throughout the set. Do not swing the torso, lean backwards, move the elbows forwards to shorten the movement, or use momentum to lift the weight.

Recommended starting working weight for an average recreational gym-goer with previous resistance-training experience: 7.5\,kg dumbbells.

\subsubsection{Face Pull}

Work: 2$\times$10

Target effort: Finish each set with approximately three repetitions in reserve (3 RIR).

Progression: Move to the next listed cable-machine load when every set finishes with 4 RIR or more.

Retention: Keep the same cable-machine load when every set finishes with approximately 1--3 RIR.

Regression: Move to the previous listed cable-machine load if the prescribed repetitions cannot be completed with the intended movement pattern and control.

The listed weight refers to the total kettlebell load attached to the 1:1 pulley using a rope attachment. The loading column gives one valid combination using the available kettlebells.

Technique: Stand or kneel far enough from the pulley that the cable remains under tension throughout the set. Begin with the arms extended and the shoulders allowed to reach forwards slightly. Pull the rope towards the face or forehead with the elbows high, allowing the hands to separate near the end of the movement. Finish with the upper arms roughly in line with the shoulders and the shoulder blades moving naturally around the rib cage. Return under control until the arms are extended again. Do not turn the movement into a pulldown, curl, triceps extension or back-extension exercise.

Recommended starting working weight for an average recreational gym-goer with previous resistance-training experience: 12\,kg kettlebell.

\subsubsection{Deadlift}

Work: 3$\times$5

Target effort: The final working set should finish with approximately one repetition in reserve (1 RIR). The first and second working sets may be easier.

Progression: Increase the starting working weight by one row when the final working set finishes with 2 RIR or more and all working sets are completed with acceptable technique.

Retention: Keep the same starting working weight when the final working set finishes with approximately 1 RIR.

Regression: Move down one row if technical failure occurs, or if acceptable technique is lost before all prescribed repetitions have been completed. If this happens in two consecutive sessions, move down two rows from the original failed load.

Technique: Position the bar over the middle of the feet and take a grip just outside the legs. Brace the trunk before each repetition and maintain a rigid, neutral back throughout the lift. Push the floor away while keeping the bar close to the body, allowing the hips and shoulders to rise together until standing upright. At the top, stand tall without leaning backwards or shrugging the shoulders. Lower the bar under control by hinging at the hips before bending the knees once the bar has passed them, returning the bar to the floor under control before beginning the next repetition. Do not bounce the bar, hitch the lift, round the back, or allow the bar to drift forwards away from the legs.

Recommended starting working weight for an average recreational gym-goer with previous resistance-training experience: 60\,kg.

\section{Alternative Hypertrophy Block}

Use this variant when training time is limited or when a simpler, hypertrophy-focused session is preferred.

For percentage-based lifts, do not progress beyond 90\%. If 90\% becomes too easy, continue at 90\% or return to the Main Strength Block.
